# Tron (Vite)

Simple browser implementation of the classic Tron lightcycle game using Vite.

Quick start

1. cd into the project folder
2. npm install
3. npm run dev

Controls

- Player 1: Arrow keys
- Player 2 (local): WASD

Modes

- 1 Player vs simple AI
- 2 Player local
